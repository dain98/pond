# Pond documentation

Pond is a lightweight, cross-platform social space built around small community
servers. Players gather in a shared town, talk, fish, decorate personal homes,
and eventually play other simple games together.

The immediate objective is much smaller: prove that two people can enter the
same cozy place, talk, and fish together.

## Confirmed direction

- Primary platforms: Windows, macOS, and Linux.
- Mobile, browsers, and consoles are not current targets.
- Pond is a social space first and a fishing game second.
- Every server has a shared commons with spaces such as a bar, cafe, and game
  room.
- Residents may eventually build and decorate personal homes.
- Fishing is the first activity; card and table games may follow later.
- Progression should emphasize expression and collection rather than power.

## First development target

The first meaningful success is two placeholder characters, running on
separate clients, walking smoothly around the same room. Art, fishing content,
housing, and public services should not get ahead of that proof.
